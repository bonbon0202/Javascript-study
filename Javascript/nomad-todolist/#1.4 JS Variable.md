<h1 id="js-variable">JS Variable</h1>
<p>Variable = 변할 수 있는 수.</p>
<pre><code>a = 2
a = 32
b = a - 5
</code></pre>
<p>a는 2가 될수도 있고 우리가 정의하는대로 32가 될 수도 있고,<br>
b가 a - 5라면 b는 a값에 따라 수가 변할 수 있다.</p>
<pre><code>let a = 221;
let b = a - 5;
a = 4;
console.log(b, a);    // 216 4
</code></pre>
